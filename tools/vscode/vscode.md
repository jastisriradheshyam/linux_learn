# VS Code

## Fold & Unfold
- Fold : Feature to collapse and unfold the code blocks
  - In command palette: `>fold level LEVEL_NUMBER`
  - `CTRL + SHIFT + [`
- Unfold:
  - In command palette: `>unfold`
  - `CTRL + SHIFT + ]`

## Go to
- Go to particular line
  - In command palette: `:LINE_NUMBER:CHARACTER_NUMBER`
