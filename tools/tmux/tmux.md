# tmux

- Default prefix: `ctrl + b`

### hierarchy

```
- Session
  - Windows
    - Panes
```

### Configuration

- config file path: `~/.tmux.conf`

## commands

### session management
- prefix + `d`: detach the tmux session

### pane Management
- prefix + `%`: Split vertical
- prefix + `"`: Split horizontal
- prefix + `x`: kill the current pane
- prefix + `directional arrow key`: move between the panes
- prefix, then `ctrl + directional arrow key`: resize the current pane
