# Container

- Containers are lightweight isolated runtimes with ability to share resources of the host operating system kernel.


## Advantages

- No need to separately manage dependencies
- Reduce development time


## History

- 1979 : `chroot`
- 2001-2003 : `Google Borg`
- 2007 : `cgroups`
- 2008 : `LXC`
  - By Canonical
- 2013 : Docker
- 2015 : Kubernetes
